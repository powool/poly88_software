# Poly 88 Software

This is an archive of my converted tapes. Most have been
done by me, with some I'll archive here until I can convert
my own.

I'll try to note in README files exactly what is a new
conversion and what is not.

## What is here

### BASIC

I have at least 4 or 5 different versions of BASIC. I'll try
to get them all converted, but at the moment, have only
converted BASIC A00 and BASIC BPRINT.

I have an early 8K one that I believe did not have strings.
Although it did fit on an 8K memory board, it wasn't all
that useful.

## Why another one?

Polymorphics cassette files use a simple one byte checksum,
which is not very robust.

Adding my own converted scans may help ensure the community
gets correct images from the old tapes.
