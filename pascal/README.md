# PASCOMP from Byte magazine

This was a Pascal P-code compiler that was published both
in BASIC and in Pascal.

I wrote my own version in 8080 assembler, am hope to include that
here.

In the meantime, I have the Pascal version here.

PASCOMP.txt may not have ever compiled. The source code here
has some errors, at least a few of which are tape conversion
problems (see the last few lines).

PASCOMP.CAS is intended to be loaded by the G02 assembler/editor,
which is what I used to enter several large text files back then.
